# Sentinel AI

Projet personnel autour de la prévention cyber et de la détection d’arnaques.

L’objectif est de partir d’un cas simple : recevoir un message suspect, l’analyser, repérer des signaux d’alerte et donner une recommandation claire à l’utilisateur.

## État actuel

Le projet est en cours.

Pour l’instant, j’ai commencé par une première brique de veille cybersécurité :
- récupération de flux RSS ;
- lecture et tri des informations ;
- envoi d’alertes via Discord.

Cette base me permet de travailler sur l’automatisation, la structuration des données et les notifications.

## Évolutions prévues

- analyse de messages suspects ;
- score de risque ;
- recommandations de sécurité ;
- API backend ;
- dashboard web ;
- meilleure gestion des sources de veille ;
- déploiement avec Docker.

## Stack

- Python
- SQLite
- Discord webhook
- Docker
- Git / GitHub

La stack pourra évoluer selon l’avancement du projet.

## Objectif

Ce projet me sert à progresser sur un sujet qui m’intéresse : le lien entre développement backend, automatisation, cybersécurité et infrastructure.
